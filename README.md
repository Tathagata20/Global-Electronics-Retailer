# Global-Electronics-Retailer
Sales Report on Global Electronics Retailer

## Situation
Maven Electronics, a global electronics retailer, needs a data analyst. They sell computers, cellphones, TVs, cameras, appliances, and more both online and offline.

## Assignment
Revenue has been on a downward trend since 2020, and management needs help to consolidate the data to conduct an exploratory analysis. They need someone to use the CSV files provided to build a data model and an interactive report that the management can use to explore performance.

## Objective
- Profile and prepare the data
- Build a relational data model
- Enrich and explore the data
- Build an interactive report

## Analysis Questions
- What types of products does the company sell, and where are customers located?
- Are there any seasonal patterns or trends for order volume or revenue?
- How long is the average delivery time in days? Has that changed over time?
- Is there a difference in average order value (AOV) for online vs. in-store sales?
